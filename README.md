# Rendi — Claude-inspired minimal dashboard (v2)

## Inhoud
- index.html (login demo)
- dashboard.html (overzicht + export)
- bank.html / beleggen.html / vastgoed.html / leningen.html / crypto.html

- assets/js/app.js
- site.webmanifest, robots.txt, favicon.svg

## GitHub Pages
Upload alles naar de root van je repo (zodat index.html bovenaan staat),
ga naar Settings → Pages → Deploy from branch → main + / (root).
